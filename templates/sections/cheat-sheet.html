VOP CHEAT SHEET
FR: Frame Number (Timeline Position)
INT: Interpolation (S=Smooth, L=Linear, I=EaseIn, O=EaseOut)
C: Corner (Check for sharp, non-spline turns)
POS: Position X, Y, Z (Camera Distance)
ROT: Rotation Pitch, Roll, Yaw (1 unit = full rotation). Application order set by job constant.
BP POS: Position X, Y, Z (Camera Distance) for the BiPack
BP ROT: Rotation Pitch, Roll, Yaw (1 unit means full rotation) for the BiPack. Application order set by job constant.
PG: ProjGel (Multiplies against loaded image)
CG: CamGel (Multiplies against physical exposure)
EXP: Exposure Time (Physical Shutter duration in seconds)
SD: Smear Distance (Frames traveled during exposure)
PH: Phase (0.0=Trail, 0.5=Center, 1.0=Lead)
