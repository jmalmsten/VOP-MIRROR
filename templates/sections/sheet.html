# FR INT C SRC STP POS ROT PG CG EXP SD PH
FR Master Frame: The point in the timeline this keyframe anchors to.
INT Interpolation Mode: (S)mooth, (L)inear, In, Out curve to the next key.
C Corner: Breaks the Catmull-Rom spline to create a sharp 3D path turn.
SRC Source Frame: The specific ProjMag frame number to load (-1 = Auto).
STP Source Step: How many source frames to advance per master frame (1 = normal).
POS Position: 3D Camera coordinates (X, Y, Z). Z sets distance to the projector plane.
ROT Rotation: Camera Pitch, Yaw, Roll angles in degrees.
PG Proj Gel: Physical filter simulation applied to the projected image.
CG Cam Gel: Physical filter simulation applied to the camera sensor capture.
EXP Exposure: Total shutter duration in seconds (controls motion blur length).
SD Scale Dampening: Future-proofing parameter for exponential scales. Keep at 1.0.
PH Phase: Timing offset for the exposure (0.5 centers the motion blur exactly on the frame).
